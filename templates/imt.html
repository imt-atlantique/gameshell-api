Getting started

Please follow these steps to get started with Gameshell:

  1. Please download the latest version of Gameshell here.
  2. Unzip the downloaded file.
  3. Open a terminal and navigate to the unzipped folder.
  4. Run the following command to start the game: $ bash start.sh
  5. That's it! You can now open the leader board at https://gameshell.org/dashboard?id=<SESSION_ID> with the SESSION_ID provided by your teacher.
